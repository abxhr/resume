# Resume

This is my resume.

## Generating the PDF

### Prerequisites

Install the required packages

```bash
sudo apt-get install texlive texlive-latex-extra
```

### Compile

Compile the `.tex` file with the command below

```bash
pdflatex resume.tex
```

This should created the `.pdf` file.
